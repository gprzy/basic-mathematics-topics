# Paridade de uma função

## Função par ($x^{p}$)

$$f(-x) = f(x)$$

**Exemplo 1: validando a paridade de uma função**
- $f(x) = x^2 - 3$
$$f(3) = f(-3)$$
$$3^2 - 3 = (-3)^2 - 3$$
$$6 = 6$$

A função $f(x) = x^2 - 3$ é par!

## Função ímpar ($x^{i}$)

$$f(-x) = -f(x)$$

**Exemplo 2: validando a imparidade de uma função**
- $f(x) = x^3 - 3$
$$f(3) = -f(-3)$$
$$3^3 - 3 = -((-3)^3 - 3)$$
$$24 = -(-30)$$
$$24 = 30$$

A função $f(x) = x^3 - 3$ não é ímpar!
