# Logaritmos

## Definição

$$\log_{b}{a} = x$$
$$b^x = a$$

Onde
- $a$ é o logaritmando
- $b$ é a base

---

## Condição de existência

$$
\log_{b}{a}
\begin{cases}
a > 0 \\
b > 0 \\
b \neq 1
\end{cases}
$$

---

## Exemplos

- $\log_{3}{9} = 2$
- $\log_{2}{16} = 4$
- $\log_{8}{2} = 1/3$

---

## Decorrências da definição

$$\log_{a}{a} = 1$$
$$\log_{b}{1} = 0$$
$$\log_{a}{a^k} = k$$
$$b^{\log_{b}{a}} = a$$
$$co\log_{b}{a} = - \log_{b}{a}$$

---

## Propriedades

$$\log_{c}{ab} = \log_{c}{a} + \log_{c}{b}$$

$$\log_{c}{\frac{a}{b}} = \log_{c}{a} - \log_{c}{b}$$

$$\log_{b}{a^n} = n \log_{b}{a}$$

$$\log_{b^n}{a} = \frac{1}{n}\log_{b}{a}$$

$$\log_{b}{a} = \frac{1}{\log_{a}{b}}$$

---

## Mudança de base

$$\log_{b}{a} = \frac{\log_{c}{a}}{\log_{c}{b}}$$

### Exemplos
- Expresse na base 3

$$\log_{2}{7} = \frac{\log_{3}{7}}{\log_{3}{2}}$$

$$\log_{2}{5} = \frac{\log_{3}{5}}{\log_{3}{2}}$$
