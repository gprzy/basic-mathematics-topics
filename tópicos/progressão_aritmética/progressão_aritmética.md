# Progressão aritmética (P.A.)

A sequência

$$(a_1, a_2, a_3, \dots, a_n)$$

é uma P.A. de razão $r$

- Crescente: $r > 0$
- Decrescente: $r < 0$
- Estacionária: $r = 0$
  
$$r = a_{n+1} - a_n$$

## Termo geral

$$a_n = a_1 + (n-1)r$$

- Exemplo: Calcular o 23º termo de $(3,7,11,\dots)$
$$r = 7-3$$
$$r=4$$
$$a_{23} = 3 + (22)4$$
$$a_{23} = 91$$

## Soma dos $n$ termos

$$S_n = \frac{(a_1 + a_n)n}{2}$$
